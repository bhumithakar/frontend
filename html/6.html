6. What is the ‘class’ attribute in HTML? With Example. ans. In HTML, the 'class' attribute is used to assign one or more class names to an element. The class attribute allows you to define a group or category that an element belongs to. This class name can then be referenced in CSS (Cascading Style Sheets) to apply specific styles or behaviors to all elements with the same class name. The 'class' attribute does not affect the behavior or functionality of the element itself; it is primarily used for styling and applying CSS rules.
