The background-position property in CSS is used to control the placement or alignment of a background image within its containing element. It allows you to specify where the background image should be positioned both horizontally and vertically. The property accepts values in various formats to determine the position of the background image. The syntax for the background-position property is as follows: background-position: horizontal-position vertical-position; Here's how you can use the background-position property: 1. Using Keywords: .element { background-image: url("image.jpg"); background-position: center top; } 2. Using Percentages: .element { background-image: url("image.jpg"); background-position: 50% 25%; } 3. Using Length Values: .element { background-image: url("image.jpg"); background-position: 20px 10px; } 4. Using a Combination: .element { background-image: url("image.jpg"); background-position: left bottom; }
